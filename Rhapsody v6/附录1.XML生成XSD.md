### 3.	手动格式化XSD  ###

有些数据为null值，需要将他们手动改为非空

```
<xs:element minOccurs="0" name="JIANCEJGMS">
    <xs:complexType>
       <xs:attribute name="null" type="xs:string"   use="optional" />
    </xs:complexType>
</xs:element>
```

修改为：

```
<xs:element minOccurs="0" name="JIANCEJGMS" type="xs:string" />
```
